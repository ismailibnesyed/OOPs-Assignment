Question 1: Write what is meant by operator overloading and method overriding with examples?
Answer 1: 
    Operator Overloading: 
        Means Redefine how operator works for object. Special method like __add__(), __sub__(), __mul__(), __len__(), __gt__() etc

        Example:             
            class Person():
                def __init__(self, name, age, money):
                    self.name = name
                    self.age = age
                    self.money = money

            class Men(Person):
                def __init__(self, name, age, money, height):
                    self.height = height
                    super().__init__(name, age, money)
                
                # Overload for Plus (+)
                def __add__(self,other):
                    return self.money + other.money
                
                # Overload for Multiplication (*)
                def __mul__(self, other):
                    return self.age * other.age
                
                # Overload for Substraction (-)
                def __sub__(self, other):
                    return self.money - other.money

            Rahim = Men('Abdur Rahim', 22, 5800, 65)
            Karim = Men('Abdul Karim', 21, 7000, 64)

            print(Rahim + Karim)
            print(Rahim * Karim)
            print(Rahim - Karim)

    Operator overriding: 
        overriding occure when a child declear its own of a method that already defined in the parent class.

        Example:    
            
            class Person():
                def __init__(self, name, age, money):
                    self.name = name
                    self.age = age
                    self.money = money

                def eat(self):
                    print(f'{self.name} eating Bread')

                def excercise(self):
                    print('Walking')


            class Men(Person):
                def __init__(self, name, age, money, height):
                    self.height = height
                    super().__init__(name, age, money)

                # Override
                def eat(self):
                    print('Burger, Kaschi, BBQ')
                def excercise(self):
                    print('Go to GYM')
                
            Rahim = Men('Abdur Rahim', 22, 5800, 65)
            Karim = Men('Abdul Karim', 21, 7000, 64)

            Rahim.eat()
            Karim.excercise()

class Person:
    def __init__(self, name, age, height, weight) -> None:
        self.name = name
        self.age = age
        self.height = height
        self.weight = weight

    def __str__(self) -> str:
        return f"{self.name}, Age: {self.age}"

class Cricketer(Person):
    def __init__(self, name, age, height, weight) -> None:
        super().__init__(name, age, height, weight)

    # Overloading
    def __gt__(self, other):
        return self.age > other.age


sakib = Cricketer('Sakib', 38, 68, 91)
musfiq = Cricketer('Rahim', 36, 68, 88)
kamal = Cricketer('Kamal', 39, 68, 94)

kalam = Cricketer('Kalam', 37, 68, 95)

players = [sakib, musfiq, kamal, jack, kalam]

oldest = players[0]
for player in players[1:]:
    if player > oldest:
        oldest = player

print("Oldest player is:", oldest)


3. Write down 4 differences between the class method and static method with proper examples.

Answer: 
    Class Method:
        1. Can access and modify class variable
        2. Used in operation related to class
        3. use @classmethod decorator
        4. takes class as first parameter
    Example:
    class Student:
        def __init__(self, name):
            self.name = name
        @classmethod
        def amni(cls, name):
            print(f'{name} meyeder piche ghur ghur kore')
    Student.amni('Ruddro')

    Static Method:
        1. Cannot directly access class or instance variable
        2. Used for helper function
        3. uses @staticmethod decorator
        4. takes no default perameter
    Example:
        class Student:
            def __init__(self, name):
                self.name = name
            @staticmethod
            def multiply(a, b):
                result = a * b
                print(result)
        Student.multiply(2, 8)
        

4.  Write what are getter and setter with proper examples?
Answer:
    Getter: Method Used to access or read the value of a private variable.
    Setter: Used to modify the value of a private variable.
    
    Example:

        class Student:
        def __init__(self, name, age):
            self.__name = name    
            self.__age = age

        # Getter
        def get_name(self):
            print(f'{self.__name} is {self.__age} years old.') 

        # Setter
        def set_age(self, age):
            if age > 0:
                self.__age = age
                print(f'{self.__name} is {self.__age} years old.')
            else:
                print("Invalid age")

    s1 = Student("Ruddro", 16)

    # Getter
    s1.get_name()

    # Setter
    s1.set_age(20)

5. Explain the difference between inheritance and composition with proper examples.
Answer:
    Inheritance:
        1. one class achive properties of another class.
        2. Achieve as parent and child class
        3. Represent 'is-a' relationship

        Example:
        
        class Animal:
            def sound(self):
                print('Sound makes by animal')

        class Dog(Animal):
            def dog_sound(self):
                print('Dog makes ghew ghew sound')

        d = Dog()

        d.sound()
        d.dog_sound()

    Composition:
        1. one class contains another class as object
        2. Creating object inside another class
        3. Represent 'has-a' relationship
        
        Example:
        class Engine:
            def start(self):
                print('Engine start now: ')

        class Car:
            def __init__(self):
                self.engine = Engine() # Composition
            
            def drive(self):
                self.engine.start()
                print('Car is starting now')

        c = Car()
        c.drive()
